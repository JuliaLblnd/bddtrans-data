BDDTrans Data
===============

Outil de recherche pour BDDTrans utilisant bddtrans-parser.

Install dependencies
--------------------

```bash
git submodule init
git submodule update
```

Requirements
------------

 - PHP 7.2
 - [JuliaLblnd/bddtrans-parser](https://github.com/JuliaLblnd/bddtrans-parser)
